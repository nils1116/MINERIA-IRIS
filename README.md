# Proyecto Final – Clasificación de Flores Iris

Este proyecto utiliza Machine Learning para predecir la especie de una flor Iris a partir de sus características:
- Largo del sépalo
- Ancho del sépalo
- Largo del pétalo
- Ancho del pétalo

El modelo fue entrenado con el algoritmo **Random Forest** y se implementó un dashboard interactivo usando **Streamlit**.
